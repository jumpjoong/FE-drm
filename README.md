# 서버 명령어 (홈브류 14버전 기준, db먼저 키고 스프링부트 실행할 것)

## 1. db서버시작

1-1. brew services start postgresql

## 2. db서버 정지

1-2. brew services stop postgresql

## 3. db서버 완전 정지 (백그라운드까지 제거)

1.3. brew services remove postgresql@14
